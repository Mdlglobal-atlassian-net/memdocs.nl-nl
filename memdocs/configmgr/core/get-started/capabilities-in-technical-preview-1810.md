# <a name="capabilities-in-configuration-manager-technical-preview-version-1810"></a>Mogelijkheden in Configuration Manager Technical Preview-versie 1810 

*Van toepassing op: Configuration Manager (technische preview-vertakking)*

Dit artikel bevat een inleiding tot de functies die beschikbaar zijn in de Technical Preview voor Configuration Manager, versie 1810. Installeer deze versie om nieuwe functies bij te werken en toe te voegen aan uw technische preview-site.
