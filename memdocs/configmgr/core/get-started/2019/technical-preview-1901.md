# <a name="features-in-configuration-manager-technical-preview-version-1901"></a>Functies in Configuration Manager Technical Preview-versie 1901

*Van toepassing op: Configuration Manager (technische preview-vertakking)*

Dit artikel bevat een inleiding tot de functies die beschikbaar zijn in de Technical Preview voor Configuration Manager, versie 1901. Installeer deze versie om nieuwe functies bij te werken en toe te voegen aan uw technische preview-site. 

Raadpleeg het artikel [Technical Preview](../technical-preview.md) voordat u deze update installeert. In dit artikel wordt u vertrouwd met de algemene vereisten en beperkingen voor het gebruik van een technische preview, het bijwerken van versies en het geven van feedback.
